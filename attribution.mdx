## Data Sources

<Warning>
  **Important**: SEC EDGAR MCP provides access to data from the U.S. Securities and Exchange Commission. All data is publicly available and provided by the SEC.
</Warning>

### Primary Data Source

**U.S. Securities and Exchange Commission (SEC)**
- **EDGAR Database**: Electronic Data Gathering, Analysis, and Retrieval system
- **Public Access**: All data is publicly available at no cost
- **Real-time Updates**: Filings are available shortly after submission
- **Historical Data**: Access to decades of financial filings

### Data Attribution

When using SEC EDGAR data through this tool, please include:

```text
Data provided by the U.S. Securities and Exchange Commission (SEC) 
through the EDGAR database. Accessed via SEC EDGAR MCP.
```
