SEC EDGAR MCP is an open-source Model Context Protocol (MCP) server that provides AI assistants with seamless access to SEC EDGAR (Electronic Data Gathering, Analysis, and Retrieval) financial data. Query company filings, extract financial metrics, analyze business segments, and process XBRL data - all through natural language.

## Key Features

<AccordionGroup>
  <Accordion title="Comprehensive SEC Data Access" icon="database">
    Access all SEC EDGAR filings including 10-K, 10-Q, 8-K, DEF 14A, and more. Retrieve company submissions, financial facts, and XBRL data with ease.
  </Accordion>
  
  <Accordion title="Smart Document Processing" icon="file-lines">
    Intelligently parse and extract sections from SEC filings. Stream large documents in chunks and extract specific items from annual reports.
  </Accordion>
  
  <Accordion title="Financial Analysis Tools" icon="chart-mixed">
    Compare financial metrics across periods, analyze business segments, and extract revenue breakdowns. Perfect for financial research and analysis.
  </Accordion>
  
  <Accordion title="XBRL Data Support" icon="code">
    Query XBRL frames across multiple entities and time periods. Discover company-specific XBRL concepts and parse XML filings including Forms 3, 4, and 5.
  </Accordion>
  
  <Accordion title="Insider Trading Analysis" icon="users">
    Track insider transactions, ownership changes, and trading patterns. Analyze Forms 3, 4, and 5 filings with comprehensive insider trading tools and complex analysis capabilities.
  </Accordion>
</AccordionGroup>

## Prerequisites

<Warning>
  **Important**: No registration is required, but according to [SEC API policy](https://www.sec.gov/search-filings/edgar-application-programming-interfaces), you must set a User-Agent header containing your email address for all requests. Set the `SEC_EDGAR_USER_AGENT` environment variable in the format: `"Your Name (your.email@domain.com)"`
</Warning>

<Note>
  **API Compliance**: Since this package uses the official SEC EDGAR API, you must respect [SEC API restrictions and data policy](https://www.sec.gov/search-filings/edgar-search-assistance/accessing-edgar-data). This includes rate limits and proper User-Agent identification as required by SEC policy.
</Note>

- Python 3.11 or higher
- An MCP-compatible client (Claude Desktop, Cline, Continue.dev, etc.)
- SEC EDGAR User-Agent header configuration
