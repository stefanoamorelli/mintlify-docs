SEC EDGAR MCP provides 7 specialized tools for accessing and analyzing SEC financial data. Each tool is designed for specific use cases and data types based on the actual GitHub repository.

## Available Tools

<Tabs>
  <Tab title="Core Tools" icon="gear">
    Essential tools for company data and financial concepts:
    
    **get_cik_by_ticker**
    Convert ticker symbols to CIK numbers
    
    **get_submissions**
    Get company filing history and metadata
    
    **get_company_concepts**
    Retrieve specific financial concepts for companies
    
    **get_xbrl_frames**
    Query XBRL frames across companies and time periods
  </Tab>
  
  <Tab title="Filing Tools" icon="file-lines">
    Tools for accessing and processing SEC filings:
    
    **get_filing_txt**
    Get filing content in text format
    
    **get_filing_txt_sections**
    Extract specific sections from filings
    
    **stream_filing_txt_chunks**
    Stream large filings in manageable chunks
  </Tab>
</Tabs>

## Performance Tips

<Steps>
  <Step title="Use Appropriate Tools">
    Choose the most specific tool for your needs. Use `get_company_concepts` for specific metrics rather than processing full filings.
  </Step>
  
  <Step title="Batch Queries">
    When analyzing multiple companies, use `get_xbrl_frames` for efficient cross-company queries.
  </Step>
  
  <Step title="Stream Large Files">
    For large filings (>5MB), use `stream_filing_txt_chunks` to avoid memory issues.
  </Step>
  
  <Step title="Cache Results">
    The tools implement intelligent caching for ticker-to-CIK mappings and frequently accessed data.
  </Step>
</Steps>

## Error Handling

All tools include robust error handling with descriptive messages:

```json
{
  "error": {
    "code": "COMPANY_NOT_FOUND",
    "message": "No company found for ticker: INVALID",
    "suggestions": [
      "Verify the ticker symbol is correct",
      "Check if the company is publicly traded",
      "Try using the full company name"
    ]
  }
}
```

Common error types:
- `COMPANY_NOT_FOUND` - Invalid ticker or CIK
- `CONCEPT_NOT_FOUND` - Unknown financial concept
- `FILING_NOT_AVAILABLE` - Requested filing doesn't exist
- `RATE_LIMIT_EXCEEDED` - Too many requests to SEC API
